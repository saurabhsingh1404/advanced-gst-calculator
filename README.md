# Advanced GST Calculator

A modern GST Calculator built using HTML, CSS, and JavaScript.

## Features

- GST Exclusive Calculation
- GST Inclusive Calculation
- CGST & SGST Breakdown
- Discount Calculation
- Taxable Amount Calculation
- Final Invoice Amount
- Copy Result Function
- Reset Calculator
- Mobile Responsive Design

## Live Demo

## GitHub Repository

## How It Works

1. Enter the amount.
2. Select GST Type (Inclusive or Exclusive).
3. Select GST Rate.
4. Enter Discount Percentage (optional).
5. Click Calculate.
6. View GST breakdown and final amount instantly.
